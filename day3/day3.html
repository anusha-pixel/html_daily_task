VS Code

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE.

VS Code features

This basic support includes syntax highlighting, bracket matching, code folding, and configurable snippets. Visual Studio Code also ships with IntelliSense for JavaScript, TypeScript, JSON, CSS, and HTML, as well as debugging support for Node. js.

Extentions of VS Code

Extensions in VS Code are add-ons that enhance its functionality by supporting additional languages, frameworks, tools, and features. They provide language support with syntax highlighting and IntelliSense, enable code formatting (like Prettier), integrate version control tools such as Git, allow real-time live previews (e.g., Live Server), and offer customization options like themes and icons. Extensions make VS Code more powerful and tailored to meet specific development needs, improving efficiency and user experience.

Emmets and their Abbreviations

Emmet is a powerful toolkit and abbreviation syntax used by web developers to speed up HTML and CSS coding. It allows for the shorthand entry of complex code structures, significantly improving productivity and reducing manual coding time.

Emmet Abbreviation
! Expands to the basic HTML structure, including the <!DOCTYPE>, <html>, <head>, <body>, and more.
+ Creates a sibling element. For example, div + p will create a div and a p next to each other.
> Creates a child element. For example, ul > li creates a ul element with a li child element.
* Repeats the element a specified number of times. For example, ul > li*5 creates a ul with 5 li items inside.
^ Moves the cursor to the parent element. For example, div > p^ moves the cursor to the parent div of p.
{ } Used to insert text or content inside an element. For example, div{Hello} creates a
containing the text "Hello."
